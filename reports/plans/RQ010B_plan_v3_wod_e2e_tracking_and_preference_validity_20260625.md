## 3. t* resolver + FULL-validation pre-flight (fix #5; Residual #7) + counterpart/candidate inclusion (#7 CLOSED)

**t*** = the current observation frame, identified **STRUCTURALLY** (empirical refinement 2026-06-26: WOD-E2E zeroes ALL absolute timestamps — `frame.timestamp_micros`, camera trigger/readout/pose times are all `0` — so t* is NOT a clock value). The structural split is unambiguous: `past_states` (16 ego history samples) = admissible past (≤ t*); `future_states` (20 samples) and the candidate `preference_trajectories` (each `EgoTrajectoryStates`, ~21 `pos_x/pos_y` points, **no per-waypoint timestamp field**) = the future (> t*) **by construction**. Admissible runtime observations = camera frames + `past_states`; forbidden as inputs = `future_states` / candidate positions.
**FULL-validation pre-flight (blocking, ratings-free):** parse all **479** sequences on HPC and assert per candidate-bearing scene: (a) exactly 3 `preference_trajectories` each with a `preference_score` field present (presence only — values stay sealed); (b) candidate-bearing scenes are resolved by **NONEMPTY `preference_trajectories`** (empirically only ~3 of ~1150 frames/shard carry candidates; the rated decision frame is the scene unit) — candidates are future-by-construction, **no waypoint-timestamp check** (none exists); (c) ≥1 admissible camera frame at the current frame (the past/observation side); (d) intrinsics/extrinsics resolve at t*. **Global rule:** >5% of 479 failing ⇒ study abstains (feasibility result). **Per-cluster rule:** any cluster with >20% failing ⇒ that cluster excluded from LOSO + flagged. Per-scene results logged pre-unlock. (Full parse is cheap; no sampling.)
**Counterpart selection (CLOSED, retained):** within R=50 m at t*, horizon H=5 s, min predicted TTC vs ego corridor; require ≥1.0 s pre-t* history; eligibility TTC≤4 s OR lateral route-gap≤2 m; tie-break TTC→lateral-gap→track-id; none ⇒ abstain. Candidate set = the 3 dataset futures (not generated).

## 6. Deterministic route/conflict-geometry algorithm + M3 interface (fix #6; Residual #3)

**Corridor:** ego reference line = smoothed ego past pose (≤t*) extended by the routing-command branch (left/straight/right) as a constant-curvature continuation over H; corridor width = ego width + fixed margin.
**Counterpart prediction:** CTRV (constant turn-rate & velocity) from the counterpart ≤t* track over H, with calibrated position uncertainty (§5) carried as a covariance.
**Geometry:** TTC = time to minimum predicted closest approach between counterpart path and ego corridor; lateral gap = min perpendicular distance of counterpart predicted path to corridor centerline over H.
**M3 interface (frozen contract):** inputs = {scene context, counterpart current-IPV estimate + estimability flag, candidate ego trajectory, corridor geometry, counterpart covariance}; output = human envelope + deviation = signed distance of candidate from envelope, with counterpart uncertainty propagated by Monte-Carlo over the calibrated error model ⇒ deviation point estimate + CI. **Depth budget (state-dependent):** enter M3 only if calibrated lateral uncertainty ≤ min(0.5 m,0.1·g) and longitudinal ≤ min(2.0 m,0.2·g) (g=gap at t*) AND geometry resolvable; else abstain. M2-downgrade guard: counterpart IPV not estimable ⇒ abstain (never M2-as-M3). Fallback frozen pre-unlock.

## 7. Estimability, abstention & the frozen analysis denominator (fix #11; Residual #4)

Per candidate-scene record `opportunity, estimability, support, deviation, abstain(+reason)`. **Funnel (all reported):** opportunity → estimable (IPV estimable ∧ within calibrated support) → **complete** (all 3 candidates valid deviation + RFS) → **non-tie** (top-2 RFS gap ≥ ε=1.0) = **D_primary**. The primary test (§8) runs on **D_primary**. **Minima applied to D_primary:** |D_primary| ≥ **60**; LOSO requires ≥**6 of 10** clusters each with ≥**8** D_primary scenes (post-filter, post-tie). Below ⇒ "underpowered" (not "negative"). Hard M2-downgrade guard = inspectable code assertion + audit field.

## 8. Pre-registered final test (fix #8; Residual #5)

- **Primary:** within-scene conditional logit for P(candidate = rater-top = argmax RFS) ~ M3 deviation, scene as stratum; effect = β (hypothesis β<0). **Tie rule:** scenes with top-2 RFS gap < ε=1.0 are excluded from the primary (they are not in D_primary).
- **Primary practical floor (β-only):** OR across the **pre-frozen IQR** of the deviation distribution ≥ **1.5**, where the IQR is computed on the ratings-free B1 deviation distribution over D_primary and **frozen+hashed pre-unlock**. One-sided **α=0.025**.
- **τ_b (within-scene Kendall) is ROBUSTNESS-ONLY and CANNOT satisfy PASS.** It is reported on complete scenes (incl. ties) for transparency.
- **PASS** = β<0 at α=0.025 with OR_IQR ≥ 1.5 **AND** nested gain over the §9 baseline **AND** all 5 §10 controls collapse **AND** LOSO direction-consistent in ≥80% of eligible folds with pooled significance **AND** |D_primary| ≥ min. Else bounded-negative, labeled (tracker-QA-fail / abstain-heavy / null-association / underpowered). Candidate-conditioned forecasting is sensitivity-only and cannot rescue a failed primary.
