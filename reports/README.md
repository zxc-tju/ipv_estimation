# Reports

This folder is the lightweight index area for future reader-facing report
packages.

Current report package roots:

- InterHub reports: `reports/interhub/ipv_estimation_results/`
- Onsite competition reports: `reports/onsite_competition/social_compliance_report/`
- Historical Argoverse reports remain under `argoverse/` or `archived/argoverse/`.

Keep report packages self-contained with `00_entry/`, `01_results/`,
`02_process/`, `README.md`, and `TRACEABILITY.md` when practical.

Large generated report payloads are ignored by git; track only lightweight
README/index files unless a task explicitly asks otherwise.
