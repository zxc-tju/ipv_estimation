# Onsite Competition Data

This directory contains onsite competition manifests and ignored local payloads.

- Full organization manifests: `data/onsite_competition/00_manifest/`
- All-team analysis package: `data/onsite_competition/all_teams_dataset/`
- Archived raw replay payloads:
  `archived/onsite_competition_raw_and_top5_subset_20260623/raw/`
- Archived top-five subset:
  `archived/onsite_competition_raw_and_top5_subset_20260623/top5_research_subset/`

The all-team package mirrors the top-five subset layout, but covers every
official scored team. Videos and replay archive zip files are intentionally
omitted from the all-team package.

Generated onsite reports live under
`reports/onsite_competition/social_compliance_report/`.
