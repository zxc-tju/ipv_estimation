# RQ012A Annotator Training Package

This package is for practice only. It contains fictional neutral-ID stubs and illustrative guidance for learning the codebook. It contains no real labels and no formal validation items.

## Contents

- `placeholder_training_items.csv`: fictional practice-item stubs.
- `illustrative_training_key.md`: training-only example answers for the fictional stubs.
- `training_vs_formal_separation.md`: required separation rule.

## How To Use

Use the training stubs only to practice applying the codebook definitions. Do not copy any training answer into a formal annotation template. Formal validation items must be labeled independently by each real annotator using only the assigned neutral item ID and the provided item material.

The training key is illustrative only. It is not derived from real study outcomes, prior annotations, or automated output.
