# Corrected Blind Issuance Notes

Package version: `RQ012A_BLIND_ISSUANCE_CORRECTED_v0.1`

Annotators receive neutral item IDs and the blank annotation template only. Use the row sequence in the assigned template as the review sequence. Do not consult outside materials or coordinate with another annotator while filling the template.

For each neutral item, mark each behavior field as `1` when the behavior is present and materially relevant in the provided material; otherwise mark `0`. Multiple behavior fields may be marked `1` when warranted. Fill `event_start_sec` and `event_end_sec` only when the behavior is temporally localizable; otherwise leave them blank. Use `confidence_1_to_5` for confidence in the behavioral judgment. Use `free_text_notes` only for concise observation notes.

Behavior fields: `aggressive_intrusion`, `appropriate_assertiveness`, `over_yielding_freeze`, `oscillation`, `deadlock`, `smooth_reciprocal_negotiation`, `unrelated_failure`.
