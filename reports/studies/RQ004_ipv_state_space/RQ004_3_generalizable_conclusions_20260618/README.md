# Generalizable Conclusions Report

Primary entry: `00_entry/index.html`.

This package contains the clean, claim-indexed generalizable conclusions report
for the sigma=0.1 full-dataset rerun. The HTML embeds its figures, so it can be
opened directly from disk.

Reusable result exports are in `01_results/`, including publication figure
PNGs/PDFs, the chart map, figure manifest, conclusions JSON, and evidence
summary JSON.

Process provenance is linked from `TRACEABILITY.md`. The old package README was
kept under `02_process/previous_readme/` for continuity.
